# HR Attrition Analysis Using Python

## Objective
To analyze employee attrition data using Python and identify factors influencing workforce turnover.

## Dataset
- IBM HR Attrition dataset (CSV)
- Contains employee demographics, salary, department, and attrition status

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Attrition distribution analysis
- Department-wise attrition comparison
- Salary vs attrition analysis

## Key Insights
- Attrition varies significantly across departments
- Employees with lower monthly income show higher attrition
- Attrition trends can be identified using exploratory data analysis
